## Not magic

Every guarantee in `@sotajs/ddd` comes from a straightforward mechanism:

- Auto-setters → reading `schema.shape` at definition time
- Frozen props → `Object.freeze()` after creation and after every action
- Identity comparison → `this.props.id === other.props.id`
- Invariants → called after `.create()` and after every action
- Branded types → TypeScript intersection with a literal string type

No code generation. No eval. No Proxy. No decorators. Plain JavaScript with
TypeScript types.
